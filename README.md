# ⚽ Tena Juniors App

**App de gestión de alumnos para la Escuela de Fútbol Tena Juniors**
Tena, Ecuador · Desde Junio 2025

## ¿Qué hace?
- Registro de alumnos por categoría (Sub-8 a Primera)
- Control de pagos mensuales desde Junio
- Estado de uniformes
- Fichas deportivas con conducta y aspectos a mejorar
- Notas de venta
- Funciona 100% offline en el celular

## Uso
Abre `index.html` directamente en Chrome. No necesita internet.

## Sync entre dispositivos
Usa el botón **Exportar** para generar un HTML con los datos incluidos y mandarlo por WhatsApp.
